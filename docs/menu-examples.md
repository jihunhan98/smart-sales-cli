# 메뉴 사용 예시

이 문서는 Smart Sales CLI의 주요 메뉴 사용 방법을 안내합니다.

## 1. 고객사 등록

```
메뉴를 선택하세요: 1

[고객사 등록]
고객사명: 삼성전자
담당자명: 홍길동
이메일: hong@samsung.com
등급 (A/B/C, 기본값 C): A
고객사 등록 완료: C001
```

## 2. 고객사 검색

```
메뉴를 선택하세요: 4

[고객사 검색]
검색어: 삼성
  C001 | 삼성전자 | 홍길동 | hong@samsung.com
```

## 3. 영업일지 등록

```
메뉴를 선택하세요: 7

[영업일지 등록]
고객사 ID: C001
활동 날짜 (YYYY-MM-DD): 2026-06-09
내용: 제품 소개 및 견적 제출
영업일지 등록 완료: R001
```

## 4. 결재 처리

```
메뉴를 선택하세요: 11

[영업일지 상신]
영업일지 ID: R001
상신 완료: R001

메뉴를 선택하세요: 12

[영업일지 승인]
영업일지 ID: R001
승인 완료: R001
```

## 5. 활동 요약

```
메뉴를 선택하세요: 15

[고객사 활동 요약]
고객사 ID: C001
  고객사 ID: C001
  전체 영업일지: 3건
  DRAFT: 1건
  SUBMITTED: 1건
  APPROVED: 1건
  REJECTED: 0건
